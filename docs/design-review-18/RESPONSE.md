the repair. C7-03 owns a stricter operational rule: capture and persist the
bounded structured line before container removal.

one-attempt pristine run. C7-03 must retain the `3/2/0` history rather than
summarizing the job as first-attempt success.

volume, database identity, direct hashes, and package state prove no data
change, and every subsequent single-service command used `--no-deps`.
C7-03 should use the corrected explicit command form from the start.
